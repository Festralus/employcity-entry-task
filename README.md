# LOREM IPSUM — Adaptive Landing Page

Адаптивная вёрстка по Figma-макету с использованием SCSS, BEM и нативного JavaScript.

## Стек технологий

- HTML5 (семантическая разметка)
- SCSS (структура из partial-файлов)
- Методология БЭМ
- JavaScript (кастомные элементы интерфейса)
- Responsive/Adaptive layout (от 320px)
- Поддержка Chrome, Firefox, Edge, Safari

## Требования задания (выполнено)

- Адаптивность от 320px
- Строгий БЭМ-нейминг
- SCSS с модульной структурой
- Sticky-header при скролле
- Возможность изменения количества шагов и полей формы без ломки сетки
- Оптимизация изображений (WebP)
- Платёжные системы в футере — не ссылки
- Контакты — ссылки
- Поля формы имеют корректные name=""
- Добавлены hover/active-эффекты
- Кастомный select
- Range-slider с динамическим обновлением процентов
- Кнопка прикрепления файла с изменением текста
- Логика слайдера шагов

## Функциональность

### Кастомный select

- Открытие и закрытие выпадающего списка
- Переключение активного значения
- Изменение состояния стрелки при открытом dropdown

### Range slider

- Живое обновление процентов
- Плавная анимация при изменении
- Устранено срабатывание жестов браузера при перетягивании

### File input

- Меняется текст кнопки при выборе файла
- Меняется иконка

### Steps

- Переключатели выровнены по центру
- Точки между элементами автоматически скрываются у первого/последнего

### Sticky header

Работает корректно во всех современных браузерах.

### Особенности реализации

- Без UI-библиотек
- Только flex, grid и clamp
- Компонентный SCSS
- Семантическая HTML-разметка
- Оптимизированные изображения
- Код структурирован для масштабируемости
